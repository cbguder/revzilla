# Revzilla

Scrapes price and stock information from Revzilla

## Usage

```
$ revzilla https://www.revzilla.com/motorcycle/shoei-rf-1400-helmet
+-------------------------------------------------------------+
| Shoei RF-1400 Helmet                                        |
+-------------------+---------+---------+---------+-----------+
| Color / Size      | MSRP    | Retail  | Members | In Stock? |
+-------------------+---------+---------+---------+-----------+
| Black / XS        | $579.99 | $579.99 | $579.99 | true      |
| Black / SM        | $579.99 | $579.99 | $579.99 | true      |
| Black / MD        | $579.99 | $579.99 | $579.99 | true      |
...
```
